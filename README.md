# Projet-4 - Billet simple pour l'Alaska
Accessible à l'adresse http://www.dev-alaa.fr

Une application CRUD pour un écrivain à l'aide de PHP et sans framework

Il s’agit du Quatrième projet de ma formation. Il a pour but la mise en pratique des concepts de la programmation orientée objet et de l’architecture MVC. Un écrivain souhaite publier son prochain roman : « Billet simple pour
 l’Alaska » sous forme de blog, chapitre par chapitre, avec pour contrainte de n’utiliser aucun CMS. Deux types d’accés sont prévus : utilisateur anonyme et administrateurs. Les utilisateurs anonymes doivent pouvoir commenter les articles, et répondre à d’autres commentaires sur deux niveaux. Le/les administrateurs peuvent ajouter, éditer et supprimer les articles. Les commentaires peuvent également être modérés.

Défis relevés
Conception d’un site de A à Z .
Implémenter toutes les fonctionnalités d’ajout, d’édition et de suppression (CRUD) en fournissant une interface agréable pour les utlisateurs.
Respect des bonnes pratiques de programmations dans un style orienté objet.
Respecter une architecture Modèle - Vue - Controlleur stricte.
